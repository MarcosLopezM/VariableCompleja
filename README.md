# Variable Compleja

Este repositorio contiene todo el material generado durante el curso de variable
compleja. El material lo conforman tareas opcionales, notas diarias escritas con 
el contenido de memoria que sirven como repaso de lo visto en clase, complementado
con algunas definiciones necesarias, así como uno que otro documento extra que vaya
surgiendo durante el curso, e.g. documentos para llevar un conteo con los ejercicios de
las guías para cada uno de los examenes parciales.

> [!NOTE]
> La mayoría de funcionalidades que aparecen en los documentos de Typst fueron
> realizadas usando modelos de IA, ya que no soy en hábil en este lenguaje de marcado y
> su sintaxis no termina de convencerme. Su uso es para documentos misceláneos.
